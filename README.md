# 🧪 SauceDemo — QA Manual Testing Project

Projeto de testes manuais realizado no sistema SauceDemo com foco em documentação QA, execução de casos de teste e reporte de bugs.

O objetivo deste projeto é demonstrar habilidades práticas de QA Manual através da validação dos principais fluxos da aplicação, utilizando documentação estruturada e evidências reais de teste.

---

# 📌 Sobre o projeto

O SauceDemo é uma aplicação web utilizada para prática de testes manuais e automação.

Neste projeto foram realizados testes nos principais fluxos do sistema, incluindo:

- Login
- Navegação de produtos
- Carrinho
- Checkout
- Logout
- Validações de formulário
- Regras de negócio

---

# 🎯 Objetivos

- Praticar testes manuais
- Criar documentação QA profissional
- Desenvolver raciocínio de testes
- Exercitar identificação e documentação de bugs

---

# 🧩 Funcionalidades testadas

## Login
- Login com credenciais válidas
- Login com usuário inválido
- Login com senha inválida
- Login com campos vazios

## Produtos
- Visualização de detalhes
- Ordenação de produtos
- Navegação entre páginas

## Carrinho
- Adicionar produtos
- Remover produtos
- Persistência do carrinho
- Continuação das compras

## Checkout
- Checkout válido
- Checkout com campos vazios
- Cancelamento de checkout
- Finalização de compra

## Logout
- Logout pelo menu lateral
- Retorno para tela de login
